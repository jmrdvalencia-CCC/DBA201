🗂️ Data Classification 🔒 Encryption 💾 3-2-1 Backup 👥 Access Reviews
🏛️ Framework
The Four Pillars of Secure Cloud Storage
Every accounting team must implement all four pillars
🗂️
1. Classify
Know what data you have and how sensitive it is before storing it anywhere
🔒
2. Encrypt
Ensure data is encrypted at rest and in transit — verify in your cloud agreements
💾
3. Back Up
Follow the 3-2-1 rule. Enable version history. Test recovery annually
👥
4. Review
Conduct quarterly access reviews. Revoke access immediately upon offboarding

⚠️ For Highly Confidential data (M&A, board financials), use client-side encryption before uploading — Azure Information Protection or Google Workspace DLP.

👥 Pillar 4: Review
Access Reviews & Employee Offboarding
Every day of delayed access revocation is a window of unauthorized access
📅 Quarterly Access Review Process
1Export list of all users with cloud access
2Verify each user still needs that access level
3Remove or downgrade unnecessary permissions
4Document review for audit trail & compliance
🚪 Employee Offboarding Checklist
Revoke all cloud platform access on last day
Transfer file ownership to manager or successor
Invalidate all active sharing links they created
Remove from Teams channels and Slack workspaces
Archive their data per company retention policy
