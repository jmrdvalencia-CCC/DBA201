🎯 The Rule: "Give users only the minimum level of access they need to perform their job — nothing more."
✅ External Auditor? → Viewer only on specific reports
✅ Junior Accountant? → Current month files only
✅ Client? → Their own invoices only
⚠️ When in doubt: Assign Viewer first. Escalate only with documented business reason.

⚙️ Platform Settings
Safe Sharing Settings Per Platform
Each tool has specific settings you must configure correctly
💬
Microsoft Teams & SharePoint
Set channels to Private by default
Apply Sensitivity Labels (Confidential)
Set link expiration dates externally
Require guest authentication
Use permission groups, not individuals
📁
Google Drive & Workspace
Share with specific emails only
Set links to Restricted by default
Use expiring access for externals
Enable Download/Print restrictions
Audit "Shared with me" regularly
⚡
Slack
Use private channels for finance
Never share raw files in Slack
Enable DLP policies
Review workspace members monthly
Use EKM for sensitive orgs
🗂️
OneDrive
Use "Specific people" sharing only
Set expiration on all shared links
Enable password-protected links
Use Personal Vault for sensitive docs
Monitor sharing activity in admin
✅ Safe Sharing Checklist
Before You Share Any Financial File
Click each item to check it off — run through this every time
Am I sharing the minimum needed — a file, not a folder?
Is the permission level set to the lowest that still works?
Have I set an expiration date on the sharing link?
Am I sharing with specific people, not "anyone with the link"?
Does the recipient still need access after this task is done?
Is this an approved company tool — not a personal account?
