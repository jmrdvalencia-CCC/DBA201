The golden rule: give only what is needed, nothing more

🔐 The Golden Rule

Principle of Least Privilege (PoLP)

"Give users only the minimum level of access they need to perform their job — nothing more."

  • Every extra permission is a potential security risk
  • Default to the lowest access level that still works
  • Escalate only with a documented business reason

💬 Microsoft Teams & SharePoint

Platform-Specific Safe Sharing Settings

  • Set channels to Private unless broad access is genuinely needed
  • Apply Sensitivity Labels (Confidential, Internal) to files
  • Set link expiration dates when sharing externally
  • Enable guest access controls — require guests to authenticate
  • Use SharePoint permission groups instead of individual user permissions
📁 Google Drive & Workspace

Safe Sharing in Google Drive

  • Share with specific people (email addresses) — never "Anyone with the link" for sensitive files
  • Set link default to Restricted in Google Admin settings
  • Use expiring access for external collaborators
  • Enable Download, Print, Copy restrictions for highly sensitive documents
  • Regularly audit "Shared with me" and "Shared drives"
⚡ Slack & OneDrive

Safe Sharing Settings

⚡ Slack

  • Use private channels for all financial discussions
  • Avoid sharing raw files in Slack — use cloud storage links with proper permissions
  • Enable DLP policies to detect sensitive data in messages
  • Regularly review workspace members and remove inactive users

🗂️ OneDrive

  • Always use "Specific people" sharing option for sensitive files
  • Set expiration dates on all shared links
  • Enable password-protected links for external sharing
  • Use Personal Vault for the most sensitive documents

⚠️ Slack messages in public channels are searchable by all workspace members. Treat public Slack channels like a company-wide bulletin board — never post sensitive data there.

✅ Quick Reference

Safe Sharing Checklist

Run through this every time you share a file containing financial, payroll, or client data:

  • Am I sharing the minimum needed — a file, not a folder?
  • Is the permission level set to the lowest that still works?
  • Have I set an expiration date on the sharing link?
  • Am I sharing with specific people, not "anyone with the link"?
  • Does the recipient still need access after this task is done?
  • Is this an approved company tool — not a personal account?
