🔑 Best Practice 1: Change Default Router Credentials

The Problem

  • Every router ships with factory credentials
  • Defaults are publicly listed on sites like routerpasswords.com
  • Common defaults: admin/admin, admin/password, admin/1234
  • Attackers use automated tools to scan and try these instantly

How to Fix It

  • Open browser → go to 192.168.0.1 or 192.168.1.1
  • Log in with current credentials (check sticker on router)
  • Navigate to Administration → Password
  • Set a strong, unique password (12+ characters)
  • Change the admin username too, if possible
🏢 Accounting Scenario: An attacker accesses your router, changes DNS settings so that bir.gov.ph resolves to a fake BIR site. You enter your firm's eFPS credentials — attacker now files fraudulent tax returns.
