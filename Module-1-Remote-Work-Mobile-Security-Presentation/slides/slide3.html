🎯 Intended Learning Outcomes

By the end of this module, you will be able to:

  • ILO 1: Identify vulnerabilities in home and public networks
  • ILO 2: Implement secure remote work practices for accounting environments
  • ILO 3: Explain the purpose and mechanics of Mobile Device Management (MDM)
  • ILO 4: Secure mobile devices against unauthorized access and data theft

📋 Note: This module focuses on remote environments — home networks, public Wi-Fi behavior, MDM, and mobile threats. It builds on (but does not repeat) the network fundamentals, malware, and insider threat concepts from your midterm modules.
